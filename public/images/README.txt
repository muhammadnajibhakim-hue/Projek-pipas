LOKASI PENYIMPANAN GAMBAR & ASET VISUAL (WEBPRENEUR)
===================================================

Gunakan folder ini (`/public/images/`) untuk menaruh gambar bisnis Anda secara terpisah seperti:
1. Logo perusahaan / sekolah
2. Foto-foto tim Kelompok 1 PIPAS
3. Tangkapan layar proyek portofolio
4. Gambar banner penunjang

CARA PENGGUNAAN PADA KODE:
--------------------------
Semua gambar yang ditaruh di dalam folder `/public/images/` ini dapat diakses secara instan di dalam HTML (`index.html`) maupun JS (`src/main.js`) Anda menggunakan path relatif yang bersih:

Contoh:
<img src="/images/nama-file-gambar.png" alt="Deskripsi" />

Rekomendasi Format:
- Gunakan .png untuk logo transparan.
- Gunakan .jpg / .webp untuk foto tim yang terkompresi hemat kuota.
- Gunakan .svg untuk logo vector yang tajam pada semua rasio screen.
